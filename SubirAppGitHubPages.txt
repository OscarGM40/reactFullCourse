Para subir una App a GitHubPages puedo hacerlo desde el package.json
Tendré que instalar en modo DEV el modulo gh-pages:
>>npm install gh-pages --save-dev 
En el package.json debo agregar la propiedad homepage:
The homepage property with its value being the string http://{username}.github.io/{repo-name} , where username is your GitHub username and {repo-name} is the name of your app’s Github repository. So in my case, its https://yajassardana.github.io/demo-app.
In the existing scripts property, we need to add the predeploy and deploy script fields:


"scripts": {
2
    "predeploy":"npm run build",
3
    "deploy":"gh-pages -d build"
4
}
